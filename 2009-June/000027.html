Então, ai você fez errado...
Você poderia ter usado o uupdate [0] ou o dch que é um programa no pacote
"devscripts"; mas cada um deles tem suas particularidades...

[0] http://debian.org/doc/maint-guide/ch-update.pt-br.html

Essas não são as melhores maneiras, já que o gpt tá no SVN e podemos
usar o "svn-buildpackage" pra gerar o pacote. Dá uma pesquisada e
qualquer dúvida me fala.. Eu tô mantendo a versão que esta no debian em Git.
Porque acho o Git muuuuito melhor que o SVN. Mas posso manter também
no SVN, se for o caso...

pra baixar:
git clone  git://git.debian.org/git/users/metal-guest/gpt.git

ou via WEB:
http://git.debian.org/?p=users/metal-guest/gpt.git;a=summary

Não sei se essa, é a lista pra gente discutir essas coisas sobre o pacote;
mas podemos?! hehe
