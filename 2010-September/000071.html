Aí depende do contexto. ;)

Sem olhar nenhum referencia agora, Portugol é a forma que nós, falantes de
portugues usamos para escrever (com papel e lapis) algoritmos.
Só que essa forma não é única, cada autor usa uma sintaxe própria, por
exemplo. Compare Manzano com outro.

E existem alguns implementação em que é possível executar no computador
esses algoritmos, o G-Portugol é um deles. ;)
Ao mesmo tempo é uma das formas de Portugol, e o projeto que fornece um
compilador para esta forma (dialeto)
